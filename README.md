Animated Palettes
=================

Installation
------------

``` r
devtools::install_github("Eomesodermin/animated")
```

Usage
-----

``` r
library("animated")

# See all palettes
names(animated_palettes)
#> [1] "A_Bugs_life" "Aladdin"
```
